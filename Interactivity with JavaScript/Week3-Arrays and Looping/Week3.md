## JavaScript Arrays

```js
//only return 1 node
document.getElementById('idName'); 

//may return multiple nodes
document.getElementsByTagName('p'); 
document.getElementsByClassName('thumbnail');
```

### Attributes and methods

```js
grades.length

```


### Difference from C/C++/...

```js
var info = ["Colleen", "1715", "Walnut", 44004];
```
